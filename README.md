# Janx Striper Tournament — New Site (Next.js)

## Quick start
```bash
npm install
npm run dev
```

## Deploy (Vercel)
1. Push this repo to GitHub
2. Import into Vercel
3. Deploy

## Update tournament details
Edit: `lib/siteConfig.ts`
- Dates, fees, prizes
- Register URL (CheddarUp)
- Catch submission form URL (Google Form)
- Leaderboard embed URL (published Google Sheet)

No code changes needed beyond that file.
